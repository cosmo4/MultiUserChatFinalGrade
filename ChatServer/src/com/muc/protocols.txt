User & Server

1. User --> Server

    * login / logoff
    * status

2. Server --> User

    *online / offline

3. User --> User

    * direct messages
    * broadcast messages / group messaging

    Commands

    login <user> <password>
    logoff

    msg <user> body....
    guest: "msg jim Hello World" <-- sent
    jim: "msg guest Hello World" <-- recv


    #topic <-- chat room / group chat
    join #topic
    msg #topic body...
